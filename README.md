# sales-data-analysis-dashboard
Data analysis project using Python, Excel, and Power BI to uncover business insights from retail sales data.

# OBJECTIVES

- I needed to clean and prepare raw sales for analysis.
- Perform EDA(Exploratory Data Analysis)
- Analyze customer purchasing behavious
- Evaluate regional and product performance
- Build an interactive Power BI dashboard to present insights.
  
  # DATASET
  
  Dataset is : SamplE Superstore dataset.
  this dataset consists categories like:
  - Order DATE
  - Ship Date
  - Customer Information
  - Product Details
  - Sales
  - Discount
  - Profit
  - Quantity
  - Region
  - State
  - Category
  - Sub-Category
    
    # TECHNOLOGIES I USED
    
    - Jupyter Notebook
    - Python
    - Pandas
    - Matplotlib
    - Numpy
    - Microsoft Excel
    - Power BI

      # EDA
      
      -What the total sales and profits are
      - which region performs best
      - which state generates the highest revenue
      - relationship between discount and profit
      - which products generate the highest sales

      # DASHBOARD FEATURES
      
      - KPI cards
          - Total sales
          - Total Profit
          - Total Orders
    - Monthly sales trend
    - sales by category
    - sales by region
    - Top 10 region
